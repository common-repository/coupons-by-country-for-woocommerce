Coupons by Country for WooCommerce lets admins restrict coupon use by country seamlessly integrating with WooCommerce for targeted promotions.

== Description ==
Our "Coupons by Country for WooCommerce" plugin empowers administrators to tailor their coupon strategies by restricting coupon usage to specific countries. This intuitive tool seamlessly integrates with WooCommerce, allowing administrators to select the desired country at the time of coupon creation. Coupons generated using this plugin will be exclusively available for use in the selected country, ensuring targeted marketing efforts and precise control over promotional campaigns.

Key Features:

1. Country-based Coupon Creation: Easily create coupons restricted to specific countries within WooCommerce.
2. Backend Control: Administrators can select the country directly from the backend during coupon creation.
3. Targeted Marketing: Tailor promotions to specific geographic regions, maximizing relevance and effectiveness.
4. Enhanced Control: Ensure that coupons are only redeemable in desired countries, preventing misuse and optimizing promotional efforts.
5. Seamless Integration: Integrates seamlessly with WooCommerce, leveraging its robust e-commerce functionalities.

Benefits of the Plugin:

1. Customized Marketing Campaigns: Craft personalized marketing campaigns by offering exclusive discounts to customers in specific countries.
2. Prevent Misuse: Restrict coupon usage to designated countries, preventing unauthorized use and preserving promotional budgets.
3. Optimized Conversion: Increase conversion rates by offering relevant incentives to customers in their respective countries.
4. Streamlined Management: Simplify coupon management by organizing promotions based on country-specific criteria, enhancing administrative control.

== Installation ==
follow the below instructions to install this plugin to your WordPress website.

Ex.

1. Upload to the `wp-content/plugins/` directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Enjoy!
